# Swarm Mode Guide

This page has been superseded by the [Hybrid Swarm Operations Guide](hybrid-swarm-operations.md), which covers the current `attoswarm` engine and configuration schema.

For quick start:

```bash
# Initialize a swarm config
attocode swarm init .

# Or copy the example
cp .attocode/swarm.hybrid.yaml.example .attocode/swarm.yaml

# Run the swarm
attoswarm run .agent/hybrid-swarm
```

See [Hybrid Swarm Operations](hybrid-swarm-operations.md) for the full runbook.
